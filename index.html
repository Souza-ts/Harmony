Sobre o Desenvolvedor

Desenvolvedor do Harmony

Apaixonado por música e tecnologia

Sou um desenvolvedor web apaixonado por criar experiências musicais incríveis. O Harmony Player é um projeto que combina minha paixão por música com minhas habilidades em programação.

Tecnologias Utilizadas

HTML5 CSS3 JavaScript Web Audio API

Características do Player

  • Sincronização precisa de letras
  • Gradiente dinâmico baseado na capa do álbum
  • Controles intuitivos de reprodução
  • Design responsivo para todos os dispositivos
  • Barra de progresso interativa
